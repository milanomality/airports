# Airports — Qt/C++ GUI Application

GUI-приложение для работы с базой данных аэропортов (WGS-84). C++17, Qt6, Google Test.

---

## Возможности

| # | Функция | Описание |
|---|---------|----------|
| 1 | **Таблица аэропортов** | Отображение кода ICAO, широты и долготы |
| 2 | **Поиск ближайших** | Все аэропорты в радиусе *N* км от выбранного (Haversine) |
| 3 | **Кратчайший маршрут** | Дейкстра с ограничением на длину сегмента *M* км |

## Алгоритмы

| Алгоритм | Сложность | Описание |
|----------|-----------|----------|
| **Haversine** | O(1) | Расстояние на сфере, R = 6371 км |
| **findNearby** | O(V log V) | Линейный обход + сортировка |
| **findShortestPath** | O(V²) | Дейкстра с ограничением на ребро |
